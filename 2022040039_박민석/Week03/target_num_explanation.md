# 하노이의 탑 풀이 과정
## Case
### 타겟 넘버는 다음과 같이 움직인다.
1. 부호
    - 각 숫자는 부호를 가질 수 있으며, 부호는 숫자에 상관 없이 존재한다.
2. 숫자
    - 각 숫자는 0을 기준으로 더해지거나 빼질 수 있다.
3. 합
    - 각 부호를 가지는 숫자는 모두 합해져 원하는 숫자를 만드는 모든 경우의 수를 나타낸다.
4. 트리
    - 배열로 입력되는 각 숫자의 나열은 0을 root node로 하는 트리의 level로 존재한다.
5. 분기
    - 트리에서 + or -의 경우로 분기되며, 각 node는 그 결과로 이어진다. 

### 문제의 요약
1. answer은 답
2. index : tree의 level
3. total : 결과 
4. tree의 leaf node로 갔을 때 답이 나와야 하므로 index가 numbers의 길이와 같을 때만 정답확인 진행
5. 정답이면 answer에 1 추가 / 아니면 리턴
6. 다시 분기
    다음 레벨로 내려가며, +와 -로 numbers의 수를 더하거나 뺀다.
