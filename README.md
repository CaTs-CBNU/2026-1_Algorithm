# 📚 2026-1_Algorithm_Study
주에 한 번씩 모여 세미나를 진행하며, 과제로 나가는 문제의 코드를 업로드하고, 코드를 리뷰할 수 있는 공간입니다.

## 주차별 계획

| 주차 | Topic |  
|:---:|:---:|
| Week 1 | OT & 기본 입출력 |
| Week 2 | 수학 |
| Week 3 | 복잡도 & 자료구조 (스택, 큐) |
| Week 4 | 기본 정렬 알고리즘 (버블, 선택, 삽입 정렬) |
| Week 5 | 고급 정렬 알고리즘 (병합, 퀵, 힙 정렬) |
| Week 6 | 그래프 탐색 기초 (DFS, BFS) |
| Week 7 | Dynamic Programming |
| Week 8 | Greedy Algorithm |
* 중간고사와 기말고사 기간에는 진행 하지 않음!  

## 💡 알고리즘 스터디 GitHub 이용 가이드
1. 개인별 폴더를 만들어 문제 풀이 진행
2. 개인별 폴더 안에 주차별 폴더 만들기 
3. 해당 주차에 주어진 과제를 폴더 안에 업로드
4. 기간 내에 풀지 못하더라도 진행한 부분까지 공유 부탁드립니다!
